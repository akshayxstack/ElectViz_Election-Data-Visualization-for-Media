# Election Data Analysis and Visualization System

## 📊 Overview

This project aims to analyze and visualize election data using modern data analytics techniques. It transforms raw data from multiple sources into meaningful insights through interactive dashboards and structured data modeling.

## 🎯 Objectives

* Integrate election data from CSV, Excel, and APIs
* Normalize constituency, candidate, and voting data
* Design a star schema data model
* Build dashboards for trend and performance analysis
* Analyze voter turnout and demographics

## 🛠️ Technologies Used

* Power BI
* Microsoft Excel
* SQL
* Data Modeling (Star Schema)

## 📈 Features

* Historical trends visualization (vote share, turnout)
* Real-time election result tracking (mock/API-based)
* Demographic analysis (age, gender, region)
* Party and candidate performance tracking
* Interactive dashboards with filters and drill-downs

## 🧱 Data Model

The project uses a **Star Schema**:

* Fact Table: Election Results
* Dimension Tables: Candidate, Constituency, Party
